# Calculator App

## Demo 
https://joshua-2333.github.io/calculator/

A fully functional calculator web application that performs basic arithmetic operations with a clean and responsive user interface. This project focuses on JavaScript logic, event handling, and dynamic UI updates.

---

## Features

- Addition, ➖ Subtraction, ✖️ Multiplication, ➗ Division  
- Clear / reset functionality  
- Delete (backspace) support  
- Handles chained operations  
- Instant calculation updates  
- Keyboard input support *(if implemented)*  
- Responsive design  

---

## Built With

- HTML5  
- CSS3  
- JavaScript (ES6+)  

---

## Overview

This project recreates a standard calculator with a focus on logic and usability. It demonstrates:

- Handling user input and button events  
- Managing calculation state and operator precedence  
- Updating the display dynamically  
- Preventing invalid operations (e.g., division by zero)
